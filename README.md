# LSTMega

Modelo de rede neural recorrente utilizando a arquitetura *Long Short-Term Memory* &ndash; LSTM para geração automatizada de textos.

Este projeto está baseado em uma implementação da biblioteca [Keras](https://keras.io/) e foi iniciado a partir de um exemplo da utilização de redes neurais recorrentes com LSTM, onde este foi disponibilizado na própria documentação do Keras e pode ser acessado [aqui](https://keras.io/examples/lstm_text_generation/).

*Projeto em desenvolvimento*.
